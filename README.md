# react-clock

Simple and customizable analog and digital clock component for React applications.

## Installation

```bash
npm install react-clock
# or
yarn add react-clock
```

## Usage

```tsx
import { AnalogClock, DigitalClock } from "react-clock";

const App = () => (
  <>
    <AnalogClock />
    <DigitalClock />
  </>
);
```
